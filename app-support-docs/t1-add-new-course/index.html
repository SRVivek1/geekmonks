Adding new course in the application

  • This guide will help you to add a new course content, configuring menu and sub-menu etc in this tutorial / blogging application.

Step-1: Create course structure

  • Create a folder structure in _courses directory and place the index.html or index.md file inside.
  • For Example: _courses/ai-ml/llm-engineering/index.md
    • First create the directories ai-ml/llm-engineering/ inside _courses/.
    • Then place the index.md inside it.

Step-2: Copy course MD files

  • Create _topics directory beside index.md file.
    • eg. _courses/ai-ml/llm-engineering/_topics
  • Now copy all your MD files inside _topics.

Step-3: Verify and configure MD files

  • Ensure that front matter is updated with correct information in the MD file.
  • Mandatory properties:
    • inded.md file layout must be course-layout, while the MD files inside _topics will have course-content lyout.
    • layout: It must be course-content
    • permalink: It should be in following format {directory_hierarchy/{file_name}.html}.
      • eg. - permalink: ai-ml/llm-engineering/index.html
      • This it used to generate the target file.
      • This is what weâ€™ll be using later to configure in submenu links.

Step-4: Add course in menu list

  • Add course name and relative URL of course content in _data/navbar/nav_simple_menu.yml file.
    • URL: /ai-ml/llm-engineering/

Step-5: Create sub-menu file

  • Create a file to add sub-menu title and links.
    • eg. _data/courses/course-nav-menu/llm-engineering.yml

Step-6: Configure sub-menu

  • Navigate to _data/courses/course_list.yml and add a new section for your course.
  • Mandatory parameters:
    • title: AI/ML - The name displayed on Website.
    • path: ai-ml/llm-engineering - Course relative URL.
    • submenu_file: llm-engineering - name of the sub-menu file present in _data/courses/course-nav-menu/llm-engineering.yml.
    • category: ai-ml - Category name.
